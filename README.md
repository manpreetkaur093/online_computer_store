# Online Computer Store 🛒

## Description
A full-stack web platform to browse and purchase computer products. Includes user authentication, shopping cart, and admin panel.

## Technologies
- HTML5, CSS3, JavaScript
- PHP & MySQL (via XAMPP)
- phpMyAdmin for DB management

## Setup
1. Clone repo to `htdocs`
2. Import `database.sql` into phpMyAdmin
3. Update `config.php` with your DB credentials
4. Run `index.php` in browser
